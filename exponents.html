Exponents

 

The first main topic is exponents. These are represented through either numerical or variable superscripts.

 

Examples: 23, x5, 9x 

 

The power that the number/variable is raised to represents the amount of times the value is multiplied by itself. 

 

Examples: 53 = 5 * 5 * 5 = 125, x2 = x * x

 

 

Special Rules:

 

 

Multiplication

 

If two exponential number are multiplied together with the same base number, their exponents are added together

 

Example: 55 * 52 = 55 + 2 = 57

 

 

Division

 

If two exponential number are divided with the same base number, their exponents are subtracted

 

Example: 58 / 56 = 58 - 6 = 52

 

 

Zero Power

 

Any number to raised to the 0 power will equal 1

 

Example: 50 = 1

 

 

Negative Power

 

Any number raised to a negative power will be equal the reciprical of the number to the absolute value of the power

 

Example: 5-3 = 1/(53) = 1/125
