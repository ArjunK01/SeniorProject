Pemdas

 

PEMDAS, or Parenthesis, Exponents, Multiplication, Division, Addition, and Subtraction, represents the order of operations. If an equation has multiple operations, you can use PEMDAS to correctly evaluate the equation. 

 

First Step: Parenthesis

 

Second Step: Exponents

 

Third Step: Multiplication and Divison, from left to right

 

Fourth Step: Addition and subtraction, from left to right

 

 

Example: 7 + (13 * 2) / (82 + 2)1/2 - 2

 

 

First Step: Parenthesis -> (13 * 2) = 26, (82 + 2) = 66

 

Updated Equation: 7 + (26) / (66)1/2 - 2

 

 

Second Step: Exponents -> (66)1/2 = 8.12

 

Updated Equation: 7 + 26 / 8.12 - 2

 

 

Third Step: Multiplication/Division -> 26 / 8.12 = 3.2

 

Updated Equation: 7 + 3.2 - 2

 

 

Fourth Step: Addition/Subtraction -> 7 + 3.2 - 2 = 8.2
