Systems of Equations

 

 

Systems of equations refers to problems involving 2 seperate equations with 2 variables (x and y)

 

Example:

 

2x + 8y = 10

x + 2y = 4

 

To solve these problems, you must first must figure out how to cancel out 1 of the 2 variables

 

In this case, you can mulitply the bottom equation by 2

 

Updated Equations:

 

2x + 8y = 10

2x + 4y = 8

 

By multiplying the bottom equation by 2, the x terms in both of the equations match, so you can now cancel out the x variable by subtracting the two equations

 

After subtracting the equations, you end up with:

 

4y = 2 --> y = 2/4 = 1/2

 

After solving for one variable, you can plug it into one of the original equations to solve for the other variable

 

x + 2y = 4

x + 2(1/2) = 4

x + 1 = 4

x = 3

 

Special Examples

 

No solution

 

Certain systems of equations can result in no solution

 

Example:

 

9x + 6y = 20

3x + 2y = 5

 

First multiply the bottom equation by 3

 

Updated Equations:

 

9x + 6y = 20

9x + 6y = 15

 

If you subtract the bottom equation from the top, both of the variables cancel out, resulting in 0 = 5. Since this is impossible, the system of equations has no solution

 

 

Infinite Solutions

 

Certain systems of equations can result in infinite solutions

 

Example:

 

4x + 8y = 20

2x + 4y = 10

 

First multiply the bottom equation by 2

 

Updated Equations:

 

4x + 8y = 20

4x + 8y = 20

 

If you subtract the bottom equation from the top, both of the variables cancel out, resulting in 0 = 0. In this case, any infinite set of variables will work with this system of equations
